# ⛏️ Minecraft State Bot

Telegram-бот для координации игроков Minecraft-государства.
Управление задачами, складом, репутацией и варнами — всё через кнопки.

---

## 🚀 Быстрый старт

### 1. Клонируй репозиторий
```bash
git clone https://github.com/your/repo.git
cd minecraft_bot
```

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Настрой переменные окружения
```bash
cp .env.example .env
# Отредактируй .env: вставь токен бота и свой Telegram ID
```

### 4. Запусти
```bash
python main.py
```

## ✨ Функционал

### 👤 Игрок
| Раздел | Что делает |
|---|---|
| 📋 Мои задачи | Список задач с прогресс-барами и дедлайнами |
| 📤 Сдать отчёт | Отправка фото + количество → передаётся админу |
| 📦 Склад | Просмотр ресурсов государства |
| 📊 Статистика | Топ игроков по репутации |
| ⚙️ Профиль | Ник, роль, репутация, варны |

### 🛡️ Администратор
| Раздел | Что делает |
|---|---|
| 📢 Объявление | Рассылка всем игрокам |
| 📋 Задачи | Создание (FSM 4 шага), удаление |
| 👥 Игроки | Список + выдача варнов |
| ⚖️ Варны | При 3 варнах — предложение кикнуть |
| 📦 Склад | Ручное редактирование ресурсов |
| ✅/❌ Отчёты | Принятие/отклонение с авто-начислением репутации |

---

## 🗄️ База данных

| Таблица | Назначение |
|---|---|
| `users` | Игроки: ник, роль, репутация |
| `tasks` | Задачи: тип, количество, дедлайн |
| `user_tasks` | Прогресс каждого игрока по задаче |
| `reports` | Отчёты с фото и статусом |
| `warehouse` | Склад ресурсов |
| `warns` | Варны с причинами |
| `logs` | Лог событий |

---

## ⏰ Авто-напоминания

APScheduler проверяет каждый час задачи с дедлайном **сегодня или завтра**
и отправляет напоминание каждому игроку с незавершённой задачей.

## 📦 Зависимости

- [aiogram 3.7](https://docs.aiogram.dev/) — фреймворк для Telegram Bot API
- [aiosqlite](https://aiosqlite.omnilib.dev/) — асинхронный SQLite
- [APScheduler](https://apscheduler.readthedocs.io/) — планировщик задач
- [python-dotenv](https://pypi.org/project/python-dotenv/) — загрузка .env
